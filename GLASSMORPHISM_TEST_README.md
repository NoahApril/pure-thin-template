## 🐛 常见问题

### Q: 玻璃效果不明显？

**A:** 可能的原因：

1. 背景是纯色，需要复杂背景才能看到效果
2. 浏览器不支持 backdrop-filter
3. CSS变量未正确加载

**解决方案：**

- 检查浏览器控制台是否有错误
- 尝试在有图案的背景上查看
- 确认在支持的浏览器中测试

### Q: 深色模式切换异常？

**A:** 检查：

1. 是否正确引入了 Tailwind CSS 的 dark 模式配置
2. CSS变量是否包含深色模式定义
3. 是否有其他样式覆盖

### Q: 动画性能差？

**A:** 优化建议：

1. 减少同时使用玻璃效果的元素数量
2. 使用 `will-change` 属性优化
3. 检查是否有不必要的重绘
